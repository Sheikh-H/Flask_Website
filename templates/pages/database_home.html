Using APIs and Databases


Throughout the course, we learn to use APIs (Application Programming Interfaces) that allow our system to connect and talk to an external system. For this requirement to be met, both systems must be available online, and access to the data source must be done through the API key.

I used MongoDB as this is free to use and online, and I can have many tables or collections. Using a NoSQL (Not Only Structured Query Language) is a first for me, as the data is stored in JSON format. With Python programmes, most of the data you use is stored or generated in JSON format, which made things slightly easier.
