Testimonial Database


This is a collection of all the 'reviews' left on my website. You can enter your own review or use the API url in a similar format to gain the same result as with the student table API function. Each result is retrieved from the database and then I use Python to calculate different information to produce different results.

I used Jinja2 templating to generate the color of each review depending on what emoji has been inserted in their feedback and also to figure out which image to use depending on whether the testimonial was left by a male or female.
