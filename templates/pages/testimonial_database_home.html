Testimonial Database


This collection of data was created to illustrate the applications of custom CSS styling for specific instances of items in the data source. On this page, each testimonial left by a person includes two pieces of data that Jinja2 Templating uses Python functions on to calibrate how the information is displayed. Namely, they are the gender and colour fields.

The gender field is populated manually by the user, but the colour field uses a similar function to the text analyser to gather whether the feedback is positive, negative, or neutral and assign a relevant colour to the testimonial. This value is then used to load a specific CSS style declared in the CSS file for each iteration of a testimonial, which changes how itâ€™s displayed.

For this function, I used emojis as the classifier for each testimonial.
