Text Analyser Function


This function was built entirely using Python and connected to the website using Jinja2 Templating. Using the form box below, you can have a piece of text analysed for the type of data it holds. The Python function calculates how many words there are, the frequency of each word, and the lexical density (unique words) of the text entered.

A High Lexical Density indicates the use of a wide variety of words, and a Low Lexical Density means high use of repeated words or phrases. The more vocabulary you know, the more you improve your ability to write with a higher lexical density.

While this function seems intriguing, it’s better to understand the different applications of this type of method. In this function, I haven’t used modules like Regular Expressions, which would make the analysis more accurate and perform complex operations like identifying a sequence of numbers or characters.
